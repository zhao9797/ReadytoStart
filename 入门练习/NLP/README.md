## 题目：利用卷积神经网络抽取实体间关系
* 参考论文：Relation classification via convolutional deep neural network.2014. COLING
* 参考框架、工具：python3.x、pytorch1.x
* 使用技术手段：word2vec、CNN、MaxPooling
* 推荐数据集：关系抽取数据集SemEval2010 task8（可以用任意其他关系抽取数据集完成）
* 推荐词向量：Glove 
* 本项目包含：
    * 1.数据集构造（预处理）
    * 2.模型搭建和训练
    * 3.模型测试和展示
